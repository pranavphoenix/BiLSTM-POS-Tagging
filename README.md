# BiLSTM-POS-Tagging
## Using Bi-LSTM to learn to tag the Parts of Speech in a sentence using NLTK Brown corpus Dataset.

#### Corpus : Brown
#### Tag set: Universal 
#### No. of words to be tagged : 1161192
#### No. of tags in the Corpus : 12

### 5 fold Cross Validation Accuracy : 96.67 %
